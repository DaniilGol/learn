Примеры использования основных свойств блочных элементов

margin используется для задания внешних отступов для элемента, может использоваться для того чтобы поставить элемент по центру экрана, как показано в этом примере
padding используется для задания внутренних отступов для элемента
border используется для задания границ для элемента
border-radius используется для задания границ для элемента
border-radius также может использоваться для создания круглых элементов
border также может использоваться для создания колонок
