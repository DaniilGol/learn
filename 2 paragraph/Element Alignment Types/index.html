Существует шесть видов выравнивания элементов:

При помощи margin-left

width 70%, margin-left 15%

При помоши отступов margin-left и margin-right

margin-right 15%, margin-left 15%

При помощи margin со значением auto

width 70%, margin 0 auto

При помощи атрибута align

div с атрибутом align="center", width 70%

С абсолютным позицированием слоя

C шириной слоя в пикселях

position absolute, width 900px, left 50%, margin-left -450px

С шириной слоя в процентах

position absolute, width 70%, left 50%, margin-left -35%
