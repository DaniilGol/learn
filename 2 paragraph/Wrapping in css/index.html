Обтекание в CSS

За обтекание отвечает float: left | right | none | inherit;

Другие непозиционированные блочные элементы без float ведут себя так, как будто элемента с float нет, так как он убран из потока. Пример:
div с float
div без float
На данном примере показано, что div без float входит внутрь div с float но при этом текст видит float и обтекает его.
Также может случиться такое, что у какого-либо div внутри будет что-либо с float, в таком случае div не увидит float элемент и просто пропустит его, чтобы это исправить можно родителю задать overflow: auto; Пример:
