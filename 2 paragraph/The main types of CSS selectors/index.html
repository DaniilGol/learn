Основные виды CSS селекторов

* - соответствует любому элементу на странице
Пример:
* { padding: 0; margin: 0; }
Уберёт всем элементам на странице внутренние и внешние отступы
#X - соответствует элементу с определённым идентификатором id
Пример:
#test { padding: 0; margin: 0; }
Уберёт элементу с идентификатором test внутренние и внешние отступы
.X - соответствует элементу с определённым классом class
Пример:
.test { padding: 0; margin: 0; }
Уберёт элементу с классом test внутренние и внешние отступы
X Y - соответствует элементу удовлетворяющему определённому контексту
Пример:
div a { color:black; }
Установит чёрный цвет ссылкам находящимся внутри div
X - соответствует элементу определённого типа
Пример:
a { color:black; }
Установит чёрный цвет ссылкам
X:visited - соответствует ссылкам на которые пользователь ещё не нажимал
X:link - соответствует ссылкам на которые пользователь нажал
Пример:
a:visited { color:black; }
Установит чёрный цвет ссылкам на которые пользователь ещё не нажимал
a:link { color:blue; }
Установит синий цвет ссылкам на которые пользователь уже нажимал
X + Y - соответствует элементу идущему сразу же за указаным
Пример:
div + a { color:black; }
Установит чёрный цвет ссылке идущей сразу после div
X > Y - соответствует элементу являющимся прямым потомком указаного
Пример:
div > a { color:black; }
Установит чёрный цвет ссылке являющейся прямым потомком div
X ~ Y - соответствует всем элементам идущим после указаного
Пример:
div ~ a { color:black; }
Установит чёрный цвет ссылкам идущим после div
X[Y] - соответствует элементам имеющим указаный атрибут
Пример:
a[title] { color:black; }
Установит чёрный цвет ссылкам имеющит атрибут title
X[href="Y"] - соответствует элементам имеющим атрибут href с указаным значением
Пример:
a[href = "youtube.com"] { color:black; }
Установит чёрный цвет ссылкам имеющит атрибут href со значением "youtube.com"
X[href*="Y"] - соответствует элементам где указаное значение должно быть частью атрибута
Пример:
a[href* = "youtube"] { color:black; }
Установит чёрный цвет ссылкам где "youtube" является частью атрибута href
X[href^="http"] - соответствует элементам где атрибут начинается указаным значением
Пример:
a[href^ = "http"] { color:black; }
Установит чёрный цвет ссылкам где атрибут href начинается с "http"
X[href$="http"] - соответствует элементам где атрибут заканчивается указаным значением
Пример:
a[href$ = ".png"] { color:black; }
Установит чёрный цвет ссылкам где атрибут href заканчивается на ".png"
X[data-*="Y"] - соответствует элементам где значение указаного пользовательского атрибута является указаным значением
Пример:
a[data-filetype="img"] { color:black; }
Установит чёрный цвет ссылкам у которых пользовательский атрибут data-filetype является "img"
X[data-*~="Y"] - соответствует элементам где значение указаного пользовательского атрибута имеет значения, разделённые пробелом
Пример:
a[data-filetype~="img"] { color:black; }
Установит чёрный цвет ссылкам у которых пользовательский атрибут data-filetype имеет значение "img" разделённое пробелами
X:checked - соответствует отмеченым элементам таким как радиокнопки или чекбоксы
Пример:
input[type = radio]:checked { color:black; }
Установит чёрный цвет отмеченым радиальным кнопкам
X:after - соответствует элементу который является последним потомком указаного элемента
X:before - соответствует элементу который является первым потомком указаного элемента
Пример:
a:after { content="'" }
Добавит кавычку после всех ссылок
a:before { content="'" }
Добавит кавычку перед всеми ссылками
X:hover - соответствует элементам на которые пользователь навёлся мышью
Пример:
a:hover { color:black; }
Установит чёрный цвет ссылка на которые пользователь навёлся мышью
X:not(selector) - соответствует выбранным элементам за исключением указаного
Пример:
div:not(.test) { color:black; }
Установит чёрный цвет всем div за исключением тех у кого есть класс "test"
X::pseudoElement - соответствует указаному псевдоэлементу у выбранного элемента
Пример:
p::first-line { color:black; }
Установит чёрный цвет первой строке каждого параграфа
X:nth-child(n) - соответствует дочернему элементу основываясь на их позиции среди группы соседних элементов
Пример:
div:nth-child(2) { color:black; }
Установит чёрный цвет второму дочернему элементу у div
div:nth-child(2n) { color:black; }
Установит чёрный цвет каждому второму дочернему элементу у div
X:nth-last-child(n) - соответствует дочернему элементу с конца основываясь на их позиции среди группы соседних элементов
Пример:
div:nth-last-child(2) { color:black; }
Установит чёрный цвет второму дочернему элементу с конца у div
div:nth-last-child(2n) { color:black; }
Установит чёрный цвет каждому второму дочернему элементу с конца у div
X:nth-of-type(n) - соответствует указаному элементу основываясь на их позиции среди группы соседних элементов
Пример:
div:nth-of-type(2) { color:black; }
Установит чёрный цвет второму по счёту div
div:nth-of-type(2n) { color:black; }
Установит чёрный цвет каждому второму по счёту div
X:nth-last-of-type(n) - соответствует указаному элементу с конца основываясь на их позиции среди группы соседних элементов
Пример:
div:nth-last-of-type(2) { color:black; }
Установит чёрный цвет второму по счёту с конца div
div:nth-last-of-type(2n) { color:black; }
Установит чёрный цвет каждому второму по счёту с конца div
X:first-child - соответствует первому дочернему элементу у выбранного элемента
Пример:
div:first-child { color:black; }
Установит чёрный цвет первому дочернему элементу у div
X:last-child - соответствует последнему дочернему элементу у выбранного элемента
Пример:
div:last-child { color:black; }
Установит чёрный цвет последнему дочернему элементу у div
X:only-child - соответствует элементу являющимся единственным потомком родителя
Пример:
div a:only-child { color:black; }
Установит чёрный цвет ссылке если она является единственным потомком у div
X:only-of-type - соответствует дочернему элементу только если он единственный у родителя
Пример:
div:only-of-type { color:black; }
Установит чёрный цвет div только если он единственный у родительского элемента
X:first-of-type - соответствует первому потомку своего типа
Пример:
div p:first-of-type { color:black; }
Установит чёрный цвет первому параграфу в каждом div
